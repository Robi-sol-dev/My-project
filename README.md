## 📌 About the Project

CyberShield Toolkit is a cybersecurity web application designed to improve
password security awareness and demonstrate basic cyber defense techniques.
It combines multiple security tools into one modern dashboard-based web app.

The main purpose is to help users create strong passwords, test password
security, and understand login protection mechanisms used in modern systems.

---

## 🎯 Project Objectives

- Improve password security awareness among users
- Demonstrate cybersecurity concepts in a practical way
- Build a modern web-based security application
- Practice frontend and backend web development
- Develop problem-solving and logical thinking skills

---

## ✨ Main Features

### 1. 🔑 Password Strength Checker
- Detects weak and strong passwords in real time
- Checks symbols, numbers, uppercase, lowercase, and length
- Suggests stronger and more secure passwords

### 2. 🎲 Secure Password Generator
- Generates strong random passwords instantly
- Allows custom password length selection
- Includes symbols, numbers, and mixed characters

### 3. 🛡️ Login Security Tester
- Simulates a secure login system
- Demonstrates brute-force attack protection
- Includes login attempt counter and account lock simulation
